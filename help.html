外部播放器使用指南

HEVC/H.265 编码格式具有高压缩比的特点，其通常会和内封 ASS 字幕同时被打包在 .MKV 格式内被各大字幕组发布。番剧库内此类资源约有 30% 左右。

但根据 Can I Use 网站的统计，当前主流的浏览器除 Safari 外均不能正常的支持 HEVC/H.265 格式的视频解码。

此外，浏览器内也难以实现 ASS 字幕的播放，因此熔岩番剧库需要您使用外部播放器来播放 MKV 和部分 MP4 资源。

移动端
Android
  • VLC (安装即用)
    • VLC For Android 熔岩云盘镜像
    VLC For Android 官网 (最新的 ARM64 即可)
  • 弹弹 Play (需手动复制链接)
    • 弹弹 Play 概念版 (Android)
iOS
  • VLC（安装即用）
    App Store
使用自己的播放器
  • 您也可以“复制链接”后在自己的播放器中播放，需要支持网络串流和字幕功能，如安卓端的弹弹 Play 可加载弹幕追番，Magma 目前一直在用~
桌面端
Windows
  • PotPlayer （安装即用）
    • PotPlayer 熔岩云盘安装包镜像
    PotPlayer 官网 (需翻墙)
  • VLC (需补丁，教程见下条帮助)
    • VLC 熔岩云盘安装包镜像
    VLC 官网
MacOS
  • IINA （安装即用）
    • IINA 官网
  • VLC
    熔岩云盘(需补丁，请自行安装后前往此仓库安装补丁。)
Linux
  • VLC (需补丁，请自行安装后前往此仓库安装补丁。)

由于 VLC Media Player 未在电脑端提供官方支持浏览器调用，因此我们需要打个补丁。
（如无特殊需求您可直接使用 PotPlayer）

Windows 系统
  • 安装 VLC 本体后，在此 此处 下载 “VLC 浏览器调用支持补丁.zip”。
  • 打开 VLC 的安装根目录（默认为 C:\Program Files\VideoLAN\VLC）
  • 将压缩包解压到 VLC 的安装根目录，右击 “vlc-protocol-register.bat”，以管理员身份运行。

关于本站点

熔岩番剧库 LavaAnime

这是个人创建的一个私有番剧库，目标是自建一个高画质高质量的私有番剧动画小圈子～

本项目创立于 2021 年 4 月，至今不打算对公共开放和盈利。


此仓库是临时用于网页托管的仓库，未来的前端可能也会放在这里。

介绍

本质是一个将 BT 在线化的平台，通过自动脚本实现下载，方便自己和好友追番。同时希望在未来尝试融合弹弹 Play 和 Bangumi 的记录功能增强体验。

本项目使用到了大量来自开源或其他开放社区的软件和数据，在此表示感谢：

  • 番组数据资源来源
    • Bangumi API
    • Bangumi 条目静态数据快照（By @czy0729） Sad, jsdelivr 国区炸了
  • 一个勉强能用的前端
    • BootStrap5 , 一个知名的 CSS 框架。
    • JQuery，主要用来完成 ajax 操作，希望未来能够学会 React。
    • Dplayer，用于在网页中播放部分浏览器能够播放的资源。
  • 后端
    • Cloudreve，主要用来管理和提供存储的资源。
  • 同时感谢各大字幕组！
